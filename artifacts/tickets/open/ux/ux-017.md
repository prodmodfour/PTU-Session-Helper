## Summary

The Dim Cave preset description says "Negated by Darkvision" and the Dark Cave preset description says "Negated by Blindsense." While true, these are misleadingly narrow. Per RAW, both Darkvision and Blindsense negate both Blindness (Dim Cave) and Total Blindness (Dark Cave). A GM reading only the preset description might incorrectly believe Darkvision is insufficient for Dark Cave.

## Suggested Fix

Update descriptions to mention both capabilities:
- Dim Cave: "Negated by Darkvision or Blindsense"
- Dark Cave: "Negated by Darkvision or Blindsense"

## Impact

Low. The code correctly negates penalties regardless of which capability is active. This is a cosmetic text issue that could cause GM confusion but does not produce incorrect game values.
