## Summary

In `gm/pokemon/[id].vue` (lines 418-421), evolution ineligibility reasons are joined with `\n`. The original `alert()` rendered these as line breaks, but the toast container uses `<span>` without `white-space: pre-line`, so newlines collapse to spaces. Multi-species ineligibility reasons run together.

## Suggested Fix

Change `.join('\n')` to `.join('; ')` so reasons render naturally as inline text in the toast.

## Impact

Low — cosmetic issue affecting evolution ineligibility messages with multiple species.
